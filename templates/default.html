Welcome to the Renewable Energy REST Web Application!

Project Information

This project consists of a REST web application that provides the client with the ability to retrieve information about developments related to renewable energy production for and across countries. The application uses an existing webservice and an own data-centric webservice to gather and expose data via endpoints. The service allows for notification registration using webhooks and is dockerized and deployed using an IaaS system.

This service does not provide any functionality on the root path level. Please try one of the paths below:

Status

Indicates the availability of all individual services this service depends on. Also, information about the registered webhooks and the uptime of the service.

Notifications

Users can register webhooks (single or multiple) that are triggered by the service based on specified events.

History

This endpoint returns historical percentages of renewables in the energy mix, including individual levels, and mean values for individual or selections of countries.

Current

This endpoint returns the latest percentages of renewables in the energy mix.
